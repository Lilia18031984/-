# 💱 Currency Converter

Приложение для конвертации валют с графическим интерфейсом на Tkinter, использующее внешний API для получения актуальных курсов.

## 👤 Автор
**Иван Иванов** (замените на свои данные)

## 📋 Описание

Currency Converter — это десктопное приложение, которое позволяет:
- Конвертировать суммы между различными валютами
- Получать актуальные курсы валют через API
- Сохранять историю всех конвертаций
- Загружать историю между запусками программы

## 🔑 Как получить API-ключ

### Шаг 1: Регистрация на exchangerate-api.com

1. Перейдите на сайт [https://www.exchangerate-api.com](https://www.exchangerate-api.com)
2. Нажмите "Get Free API Key"
3. Введите ваш email и нажмите "Subscribe"
4. Подтвердите email в письме

### Шаг 2: Получение ключа

После подтверждения вы получите API-ключ вида: `abcdef1234567890abcdef`

### Шаг 3: Настройка приложения

1. Запустите приложение
2. Вставьте API-ключ в поле "API Key"
3. Нажмите "Сохранить API Key"
