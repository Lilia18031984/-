# 🐙 GitHub User Finder

Приложение для поиска пользователей GitHub с графическим интерфейсом на Tkinter.

## 👤 Автор
**Ткаченко Дмитрий** 

## 📋 Описание

GitHub User Finder — это десктопное приложение, которое позволяет:
- Искать пользователей GitHub по имени
- Просматривать информацию о пользователе (аватар, имя, количество репозиториев, подписчиков)
- Добавлять пользователей в избранное
- Сохранять список избранных пользователей между запусками программы
